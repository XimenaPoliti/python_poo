# Project python_poo
Proyecto creado con el fin del aprendizaje de la Programación Orientada a Objetos enfocado al lenguaje python

### En este curso he aprendido
- Identificar cómo Python interpreta y ejecuta instrucciones
- Estructurar programas utilizando la Programación Orientada a Objetos
- Elementos de POO
- Pilares: Encapsulamiento, Abstracción, Herencia y Polimorfismo con python
- Decomposición y Modularidad
- Clases: instancia, constructor, variables y métodos
- Decoradores
